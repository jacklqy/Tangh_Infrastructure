# Tangh_Infrastructure
一个基础性的项目，主要是一些帮助文件

其中包含，自定义控件，邮件，加密，Ini文件，Http请求，Http监控，图片处理，设备码随机，IP管理，
日志管理，拼音，代理，短信，套接字，Sql，时间，VPN，Win32Api等处理。
